## 目标
- 页面初始化时，将相机/控制器默认对准场景正前方（原点），呈现正面视角。
- 不改变现有交互（仍可旋转），只修正初始朝向；如需锁定为永远正面，可作为可选增强。

## 修改点
1. 更新 OrbitControls 初始目标
- 位置：`src/components/ParticleScene.tsx:405-406`
- 变更：将 `controlsRef.current.target.set(0, 2, 0)` 改为 `controlsRef.current.target.set(0, 0, 0)`，保持 `update()` 不变。
- 说明：当前目标 Y=2 会让相机初始略微上仰；改为原点即可正面对齐。

2. 保持相机位置
- 位置：`src/components/ParticleScene.tsx:410`
- 保持 `camera={{ position: [0, 0, 80], fov: 60 }}` 不变；该位置配合目标(0,0,0)即为标准正面视角。

## 可选增强（按需开启）
- 若希望“仅初始化正面，但允许后续自由旋转”，无需额外改动。
- 若希望“锁定为始终正面（不允许抬头/低头）”：在 `useEffect` 中设置
  - `controlsRef.current.minPolarAngle = Math.PI / 2`
  - `controlsRef.current.maxPolarAngle = Math.PI / 2`
  - 这样可将极角固定在水平面，实现严格正面视角。也可只缩窄范围，例如 `minPolarAngle = Math.PI/2 - 0.2`, `maxPolarAngle = Math.PI/2 + 0.2`。

## 验证要点
- 启动开发服务后，刷新页面应直接看到粒子云“正面”展示，画面不再上仰。
- 旋转操作仍正常（除非开启了极角锁定增强）。

## 影响范围
- 仅调整控制器目标，不影响粒子逻辑、着色器或后期处理。

请确认按“仅修正初始朝向”执行，或选择加入“极角锁定”增强。
