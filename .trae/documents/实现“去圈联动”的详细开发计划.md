## 目标
- 默认不显示中心圆圈；提供一键“无圈展示”预设与总开关，随时恢复。
- 从动力学、着色器、后期三层联动，消除半径附近堆积带与亮边。

## 文件改动
### src/components/ParticleScene.tsx
- 新增 props：`ringBand:number`、`ringWidth:number`、`blendMode:'additive'|'normal'`。
- 动力学：在积分循环加入带状斥力（ring band）：
  - 计算半径 `d = sqrt(px*px + py*py + pz*pz)`；若 `abs(d - voidRadius) < ringBand`，施加排斥
  - 力度 `force = (ringBand - abs(d - voidRadius)) / ringBand * antiCenterStrength`，方向按 normalize([px,py,pz])，系数 0.04（与现有步长一致）
- 着色器：在片元中新增 `uRingWidth`，计算 `ringFall = 1.0 - smoothstep(uVoidRadius, uVoidRadius + uRingWidth, d)`；`alpha *= ringFall`，圈内继续 `discard`
- 材质混合：根据 `blendMode` 切换 Points/Shader 两种材质的 blending（Additive/Normal），保留 depthWrite=false；保留 DPI uniform

### src/components/TunePanel.tsx
- 新增滑块：
  - `带状斥力范围 ringBand`（范围 3–10，默认 6）
  - `环形衰减宽度 ringWidth`（范围 4–12，默认 8）
- 新增开关：`混合模式`（加法/普通）
- 新增总开关：`隐藏中心圈`（一键关闭斥力/遮罩，半径/强度归零，并提升 Bloom 阈值到 0.6，可恢复）
- 保持现有控件：斥力半径/强度、遮罩半径/软硬、Bloom 阈值/强度、粒子尺寸、粒子数量、FPS、截图

### src/App.tsx
- 新增状态：`ringBand`、`ringWidth`、`blendMode`
- 将新状态传入 Scene 与 TunePanel；为 TunePanel 的总开关提供回调（更新 antiCenterEnabled/voidMaskEnabled/antiCenterRadius/antiCenterStrength/bloomThreshold）
- 预设新增“无圈展示”：
  - `antiCenterEnabled=false`、`voidMaskEnabled=false`、`antiCenterRadius=0`、`antiCenterStrength=0`
  - `ringBand=6`、`ringWidth=8`、`blendMode='normal'`
  - `bloomThreshold=0.6`、`bloomIntensity=0.6`、`pointSize=0.18`
- 默认进场应用“无圈展示”（useEffect 初始化）

## 验证

- 调整 ringBand/ringWidth 与 Bloom 阈值，圈保持不可见；切换混合模式做 A/B
- 保持 FPS 稳定（30k 粒子桌面 >50FPS），交互无异常
